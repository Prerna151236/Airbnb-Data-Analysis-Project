📌 Project Overview

This project analyzes Airbnb listing data to uncover insights about pricing trends, listing distribution, occupancy performance, and customer review behavior. The goal is to transform raw data into meaningful insights using Python for data cleaning and exploratory data analysis, and Power BI for interactive data visualization.

The dashboard helps identify high-performing listings, profitable room types, and neighborhoods with the highest demand.

🎯 Business Problem

Short-term rental platforms like Airbnb generate large amounts of listing and customer data. Hosts and stakeholders need insights to understand:

Which neighborhoods have the highest listing supply

Which room types generate the most revenue

How customer reviews influence listing popularity

How occupancy impacts revenue potential

This project addresses these questions through data analysis and visualization.

📊 Dataset Information

Dataset contains Airbnb listing information including pricing, availability, property details, host information, and customer reviews.

🧹 Data Cleaning (Python)

Data cleaning was performed using Python libraries such as Pandas and NumPy.

Cleaning steps included:

Handling missing values

Removing irrelevant columns

Converting data types

Removing duplicates

Standardizing column names

Handling price formatting

These steps ensured the dataset was ready for analysis and visualization.

🔎 Exploratory Data Analysis (EDA)

EDA was performed to understand patterns and relationships within the dataset.

Key analysis included:

Price distribution analysis

Room type popularity

Review trends

Occupancy vs revenue analysis

Visualizations were created using:

Matplotlib

Seaborn

KPI Metrics

Dashboard Visualizations

Listings distribution by neighborhood

Revenue performance by room type

Occupancy vs revenue analysis

Customer review analysis

These visuals help stakeholders quickly understand market trends and listing performance.

💡 Key Insights

Some of the major insights from the analysis include:

Entire home listings generate higher revenue compared to private rooms.

Certain neighborhoods dominate the Airbnb market with the highest listing counts.

Listings with higher occupancy tend to generate higher annual revenue.

Review activity indicates strong guest engagement for popular listings.
